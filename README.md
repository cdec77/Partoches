# Partoches
Une petite application utilisable sous toutes les plateformes : Linux, Android, Windows, iOS, Mac à partir des navigateurs Chromium, Chrome ou Safari.
Elle permet d'accéder à toutes ses partitions très facilement sur son appareil et préparer des sets pour les concerts sans avoir à les copier ou les ranger.
Un outil très utiles pour les musiciens amateurs ou pro qui utilisent une tablette ou un ordinateur pour lire leurs partitions.
